# Security Policy

This project does not contain security‑sensitive components, network features, or
export‑restricted code. It is a local LVGL + SDL2 simulator intended for development and debugging
on Windows.

## Reporting Security Issues

If you believe you have found a security issue, you may open an issue on GitHub.
Please include:

- A clear description of the problem  
- Steps to reproduce
- The environment in which it occurs

Since this project is maintained in my spare time, response times may vary.
